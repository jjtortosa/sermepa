## Installation

Download node at [nodejs.org](http://nodejs.org) and install it, if you haven't already.

```sh
npm install sermepa --save
```

## Testing

cd node_modules/sermepa
$ npm test

Navegar a:
[http://localhost:3000/](http://localhost:3000/)
